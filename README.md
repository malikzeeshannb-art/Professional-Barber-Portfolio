# Malik Sufyan — Premium Barber & Grooming Specialist

A modern, responsive barber portfolio website built with **HTML**, **CSS**, and **Vanilla JavaScript**.  
This project is designed to present barber services, highlight transformation work, build trust with clients, and make booking easy through WhatsApp and direct contact links.

## Project Overview

This website is a single-page portfolio for a professional barber and grooming specialist based in Islamabad. It includes a strong hero section, service highlights, gallery showcases, testimonials, an FAQ section, and a contact area with booking options.

The goal of the project is to create a clean, premium, client-friendly online presence that works smoothly on desktop, tablet, and mobile devices.

## Features

- Responsive one-page portfolio layout
- Sticky header with navigation
- Mobile-friendly navigation toggle
- Hero section with responsive background images
- Typewriter-style headline animation
- Scroll-based active navigation highlighting
- Smooth scrolling for anchor links
- Services section for haircut and grooming offerings
- Trust section to build confidence with clients
- Gallery section for haircut transformation images
- Video testimonial section
- FAQ accordion
- Contact section with WhatsApp, phone, email, and social links
- Floating WhatsApp button for quick booking
- Responsive adjustments for smaller screens
- Clean, professional barber-themed design

## Tech Stack

- **HTML5** — page structure and content
- **CSS3** — layout, responsiveness, and visual styling
- **JavaScript (Vanilla)** — interactivity and UI behavior
- **Font Awesome CDN** — icons

## Local Run Instructions

### Option 1: Live Server in VS Code
1. Open the `sufyans-portfolio` folder in VS Code.
2. Open `index.html`.
3. Right-click and choose **Open with Live Server**.
4. Or click **Go Live** from the bottom-right corner.

### Option 2: Open Directly in Browser
Double-click `index.html` to open it in your browser.

## Deployment

This is a static website, so it can be deployed easily to:

- GitHub Pages
- Netlify
- Vercel
- Any static hosting service

### GitHub Pages
1. Push the project to a GitHub repository.
2. Go to repository **Settings**.
3. Open **Pages**.
4. Select the branch that contains `index.html`.
5. Save the settings.
6. Copy the live URL and share it with clients.

## Notes

- The project uses relative image and video paths, so keep the folder structure unchanged.
- Font Awesome is loaded from a CDN, so internet access is required for icons to display.
- The testimonial video is stored in `assets/videos/`.
- Keep `index.html`, `style.css`, and `script.js` in the root folder for smooth hosting.

## Customization

You can customize this project by editing:
- Text content in `index.html`
- Colors, spacing, layout, and animations in `style.css`
- Navigation, animations, FAQ behavior, and scroll features in `script.js`

## Contact / Booking

Primary booking is set up through WhatsApp inside the buttons and contact section.  
You can update the phone number, email address, social links, and booking message directly in `index.html`.
